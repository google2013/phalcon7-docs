Checks if a value has a correct FILE format  

.. code-block:: php

    <?php

    use Phalcon\Validation\Validator\File as FileValidator;
    
    $validator->add('file', new FileValidator(array(
     'mimes' => array('image/png', 'image/gif'),
     'minsize' => 100,
     'maxsize' => 10000,
       'message' => 'The file is not valid'
    )));
